World

The world uses a dynamic time and weather system to create an immersive world.

The world time system is highly customizable, allowing for both developer control and player game settings. Time can be set to have a certain number of minutes in a day, and what the ratio of real to in game minutes is. Anything may subscribe to the world time system to trigger delegates with trigger options like repitition. The skybox material uses a world light to project a sun and moon onto it, and the light updates along with the time

The worlds weather system subscribes to the time system to randomly change weather on a repeating, customizable interval. Weather presets are created as data, holding information for precipitation type, intensity, cloud coverage and the like. When the current weather changes, the world slowly updates as to not break immersion. The game currently supports sunny, cloudy, rainy and snowy weather with a high degree of customizability
